# Game of Life
Conway's Game of Life implemented with GLFW, OpenGL and Dear ImGui

## Features
 - Pan
 - Zoom
 - Toggle fullscreen
 - Control the simulation's FPS
 - Show metrics window (FPS and frametimes)
